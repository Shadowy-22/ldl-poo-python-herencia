# Actividad N°6 - POO en Python (Herencia)

## 📌 Descripción
Este proyecto consiste en el desarrollo de un sistema en Python utilizando el paradigma de Programación Orientada a Objetos (POO), aplicando el concepto de herencia.

El sistema permite administrar jugadores de un equipo de fútbol, almacenando información como:
- Número de camiseta
- Apellido
- Posición (arquero, defensor, central o delantero)
- Minutos jugados
- Cantidad de goles (excepto para arqueros)

---

## 🎯 Objetivo
Aplicar los conceptos de herencia en Python para modelar un sistema orientado a objetos que represente adecuadamente el escenario planteado.

---

## 🧩 Funcionalidades
- Creación de jugadores según su posición
- Registro de minutos jugados
- Registro de goles (según corresponda)
- Consulta de datos de los jugadores
- Interfaz mediante menú por consola
